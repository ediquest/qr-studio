# Realtime Barcode & QR Studio (v3.9.14)
- Stabilny build (bez svg2pdf) — mniejsze ryzyko 500 na Vite.
- Generator: stan + podgląd w localStorage, PNG multiplier ×1–10, pobieranie PNG/SVG.
- Batch: CSV + sekwencja z auto-inkrementacją.
- Etykiety: siatka i Free layout (drag po stronie), uchwyt ◢, dłuższe suwaki.
- PDF: eksport przez PNG (jsPDF).
